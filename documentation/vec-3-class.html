Vec3 类

源码

基本使用

Vec3 是用于所有 3 维向量操作的类。

创建

要创建一个新的 Vec3，只需使用它的构造函数。
如果不带参数使用，它将创建一个 X、Y 和 Z 分量等于 0 的空向量 Vec3。

// X、Y 和 Z 分量等于 0 的空向量
const nullVector = new Vec3();
const vector = new Vec3(1, 1, 1);

链式调用

由于大多数 Vec3 方法返回向量本身，因此您可以进行链式调用：

// 创建一个新的 Vec3 向量并且链式调用
const vector = new Vec3().addScalar(2).normalize();

onChange 回调

Vec3 类使用 getter 和 setter，允许每次向量分量之一发生变化时执行一个函数：

// 创建一个新的 Vec3 向量并监听其变化
const vector = new Vec3().onChange(() => {
// 向量的分量之一刚刚改变，做一些事情...
const normalizedVector = vector.normalize();
});
// 这将触发我们的 onChange 回调
vector.x = 1;
// 注意，这会再次触发 onChange！
vector.y = 2;

参数

  • X float，可选 沿 X 轴的值。默认为 0。
  • Y float，可选 沿 Y 轴的值。默认为 0。
  • Z float，可选 沿 Z 轴的值。默认为 0。

属性

  • x (float): v7.0

    沿 X 轴的值。

  • y (float): v7.0

    沿 Y 轴的值。

  • z (float): v7.0

    沿 Z 轴的值。

方法

  • add(vector): v7.0

    把一个向量与当前向量相加

    • vector Vec3 class object 需要相加的向量

    返回值：相加后的向量

  • addScalar(scalar): v7.0

    给每个分量分别加上一个值

    • scalar float 相加的值

    返回值：相加后的向量

  • applyMat4(matrix): v7.0

    将一个四维 Mat4 矩阵应用于此向量。

    • matrix Mat4 class object 要应用的矩阵

    返回值：应用矩阵后得到的此向量。

  • applyQuat(quaternion): v7.1

    将一个四元数（三维空间中的旋转）应用到此向量上。

    • quaternion Quat class object 要应用的四元数

    返回值：应用四元数变换后得到的此向量。

  • clone(): v7.0

    克隆此向量。

    返回值：新创建的克隆向量

  • copy(vector): v7.0

    将一个向量复制到此向量中。

    • vector Vec3 class object 要复制的向量

    返回值：复制操作后的此向量。

  • dot(vector): v7.0

    计算两个向量的点积（内积）。

    • vector Vec3 class object 用于计算点积的向量

    返回值：代表两个向量点积结果的浮点数。

  • equals(vector): v7.0

    检查两个向量是否相等。

    • vector Vec3 class object 用于比较的向量

    返回值：如果两个向量相等则返回 true，否则返回 false。

  • max(vector): v7.0

    将最大值应用到此向量上。

    • vector Vec3 class object 其他向量

    返回值：应用了最大值的新向量。

  • min(vector): v7.0

    将最小值应用到此向量上。

    • vector Vec3 class object 其他向量

    返回值：应用了最小值的新向量。

  • multiply(vector): v7.1

    将一个向量与该向量相乘。

    • vector Vec3 class object 用于相乘的向量

    返回值：相乘后的此向量。

  • multiplyScalar(scalar): v7.1

    将一个标量与该向量相乘。

    • scalar float 用于相乘的数字

    返回值：相乘后的此向量。

  • normalize(): v7.0

    规范化（单位化）此向量。

    返回值：规范化后的向量。

  • project(camera): v7.1

    将三维坐标投影到二维点上。

    • camera Camera class object. Use a plane camera. 将此向量从三维空间投影到二维空间所使用的相机。

    返回值：投影后的此向量。

  • sanitizeNaNValuesWith(vector): v7.0

    当值为 NaN 时，将此向量与另一个向量合并。主要用于内部避免错误。

    • vector Vec3 class object 用于清理操作的向量（即当原始向量值为 NaN 时，用这个向量的值替代）。

    返回值：清理过的向量。

  • set(x, y, z): v7.0

    通过给定值设置向量。

    • x float 设置 X 向量的值
    • y float 设置 Y 向量的值
    • z float 设置 Z 向量的值

    返回值：设置后的新向量。

  • sub(vector): v7.0

    从该向量中减去另一个向量。

    • vector Vec3 class object 用于相减的向量。

    返回值：减法操作后的此向量。

  • subScalar(scalar): v7.0

    从该向量中减去一个标量值。

    • scalar float 用于相减的数字。

    返回值：减法操作后的此向量。

  • unproject(camera): v7.1

    将二维点反投影到三维坐标上。

    • camera Camera class object. Use a plane camera. 将此向量从二维空间反投影到三维空间所使用的相机。

    返回值：反投影后的此向量。

事件

  • onChange(callback): v8.0

    每次向量的 x、y 或 z 分量发生变化时执行一个函数。

    • callback function 回调函数

    返回值：当前的 Vec3 对象本身
