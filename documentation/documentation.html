文档

基本概念

自7.0版本以来，整个库已被重写并分解为多个类模块，以提高代码的可读性和可维护性。由于curtains.js 是一个高级 API 库，因此大多数类仅在内部使用。即使它们列在这里，你也永远不需要使用它们。 Since the version 7.0 the whole library has been rewritten and broken into multiple class modules to improve the readability and maintainability of the code. Since curtains.js is a high-level API library, most of those classes are used internally only. Even tho they are listed here you won't ever need to use them.

如果您来自以前版本的库，请查看迁移指南。

核心

创建基本场景所需的主要类。 Main classes needed to create basic scenes.

  • Curtains v1.0

    Curtains 类负责画布和 WebGL 上下文，因此创建 WebGL 场景。它还创建一个 requestAnimationFrame 循环，并将绘制您创建的每个 Plane 元素。 The Curtains class is responsible for the canvas and WebGL context and therefore creates a WebGL scene. It also creates a requestAnimationFrame loop and will draw every Plane elements that you've created.

    源代码.

    • Renderer

      Renderer 类处理与 WebGL 上下文相关的所有内容（创建和恢复、扩展、WebGL 命令等）。 The Renderer class is handling everything related to the WebGL context (creation and restoration, extensions, WebGL commands, etc).

      源代码.

      • Scene

        Scene 类由 Renderer 创建。它管理添加的每个 WebGL 对象（平面、渲染目标和着色器通道）的渲染顺序。详细了解该库如何绘制不同的对象 The Scene class is created by the Renderer. It manages the render order of every WebGL objects (planes, render targets and shader passes) added.
        Read more about how the library draws the different objects.

        源代码.

      • CacheManager

        CacheManager 缓存已编译的着色器、顶点和 UV 数组以及图像纹理，以便尽可能重用它们并避免额外繁重的计算。 The CacheManager caches already compiled shaders, vertices and UVs arrays and images textures in order to reuse them whenever possible and avoid extra heavy computations.

        源代码.

      • CallbackQueueManager

        CallbackQueueManager 用于在下一个动画帧调用时执行回调。 The CallbackQueueManager is used to execute callbacks on the next animation frame call.

        源代码.

    • ScrollManager

      ScrollManager 类根据需要添加滚动事件侦听器并处理滚动值的更新。 The ScrollManager class adds a scroll event listener if needed and handles the update of the scroll values.

      源代码.

  • Plane v1.0

    Plane 类创建一个绑定到 HTML 元素的 WebGL 平面，并将按照该 HTML 元素的大小和位置进行渲染。 The Plane class creates a WebGL plane that is bound to a HTML element and that will be rendered at the sizes and positions of this HTML element.

    它扩展了 DOMMesh 类。

    源代码.

    • DOMMesh

      DOMMesh 类通过添加一些依赖于用于创建网格的关联 HTML 元素的辅助方法来扩展 Mesh 类。 The DOMMesh class extends the Mesh class by adding a few helper methods that relies on the associated HTML element used to create the Mesh.

      源代码.

      • Mesh

        Mesh类创建一个WebGL网格，即创建一个几何体（Geometry）、一个程序（Program），处理网格的统一变量（uniforms）和纹理（Textures）的创建。 The Mesh class creates a WebGL mesh, i.e. creates a Geometry, a Program, the Mesh's uniforms and handles Textures creation.

        源代码.

        • Geometry

          几何类（The Geometry class）负责创建并绑定顶点和纹理坐标属性的缓冲区，并绘制网格。 The Geometry class creates and binds the vertices and texture coordinates attributes buffers and draw the mesh.

          源代码.

        • Program

          程序类（The Program class）负责编译着色器并将它们链接成一个WebGL程序，这个程序将被网格（Mesh）所使用。 The Program class compiles the shaders and creates a WebGL program used by the Mesh.

          源代码.

        • Uniforms

          uniforms类（The Uniforms class）负责创建并更新网格（Mesh）的uniform变量。 The Uniforms class creates and updates the Mesh's uniforms.

          源代码.

    • Camera

      相机类（The Camera class）负责创建一个透视相机，用于计算平面的投影矩阵。 The Camera class creates a perspective camera used to compute the plane's projection matrix.

      源代码.

  • Texture v2.0

    Texture类用于创建WebGL纹理。

帧缓冲区对象

这些类通常用于为您的平面添加后期处理效果。 Those classes are generally used to add post-processing effects to your planes.

  • RenderTarget v5.0

    RenderTarget类是一个内部使用的帧缓冲对象（Frame Buffer Object），用于创建着色器通道。它允许您将一个或多个平面渲染为纹理，然后您可以在另一个平面或着色器通道中使用这些纹理。 The RenderTarget class is a Frame Buffer Object used internally to create shader passes. It allows you to render one or multiple planes as a texture that you can then use in another plane or shader pass.

    源代码.

  • ShaderPass v3.0

    ShaderPass类扩展自DOMMesh类。
    ShaderPass元素用于添加后期处理效果。可以将其视为一个附加到WebGL画布容器的额外平面，这样您就可以对一组平面或整个WebGL场景应用着色器了。
    您可以添加多个着色器通道，并且它们会按照创建顺序逐个应用。 The ShaderPass class extends the DOMMesh class.
    ShaderPass elements are used to add post-processing effects. Think of it as an additional plane that will be bound to your WebGL canvas container. This way you will be able to apply shaders to a bunch of planes or to your whole WebGL scene.
    You can add multiple shader passes and they will be applied one after the other in their creation order.

    源代码.

Loaders

可用于创建基于 HTML 媒体元素的 WebGL 纹理 Useful to create WebGL textures based on HTML media elements

  • TextureLoader v7.0

    TextureLoader 类加载图像、视频和画布，并使用这些 HTML 元素作为源来创建纹理。 The TextureLoader class loads images, videos and canvases and creates Textures using those HTML elements as sources.

    源代码.

    • PlaneTextureLoader

      PlaneTextureLoader 类扩展了 TextureLoader 类。由平面和着色器通道在内部使用来加载媒体、使用生成的纹理并触发 onLoading 和 onReady 回调。 The PlaneTextureLoader class extends the TextureLoader class. Used internally by planes and shader passes to load medias, use the resulting textures and fire the onLoading and onReady callbacks.

      源代码.

Extras

额外的类通常是使用上面的其他类构建的附加类，可以帮助您节省一些时间。 Extra classes are often used additional classes built using the other classes above that can help you save some time.

  • PingPongPlane v7.0

    PingPongPlane 类通过添加两个在渲染过程中会进行交换的读写渲染目标以及获取最终纹理的方法，扩展了 Plane 类。此类设计旨在支持流体特效、流体贴图等功能。 The PingPongPlane class extends the Plane class by adding 2 read and write render targets that will be swapped during render and a method to get the resulting texture. Allows fluid effects, flowmaps, etc.

    源代码.

  • FXAAPass v7.0updated in v8.0

    FXAAPass 类通过创建快速近似抗锯齿通道来扩展 ShaderPass 类。如果您在代码中使用渲染目标或着色器通道，只需使用此通道作为最后一个通道即可添加抗锯齿效果。 The FXAAPass class extends the ShaderPass class by creating a Fast Approximate Anti-Aliasing pass. If you're using render targets or shader passes in your code, just use this pass as the last pass to add an anti-aliasing effect.

    源代码.
