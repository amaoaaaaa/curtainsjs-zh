Vec2 类

源码

基本使用

Vec2 是用于所有二维向量操作的类。

创建

要新建一个 Vec2 对象，只需调用其构造函数即可。
如果不传递任何参数，则会生成一个 X 和 Y 分量都为 0 的零向量。

// X 和 Y 分量都为 0
const nullVector = new Vec2();
const vector = new Vec2(1, 1);

使用链式调用

由于 Vec2 类中的大多数方法都会返回该向量自身，因此您可以方便地进行链式调用：

// 创建一个新的 Vec2 向量并链式调用方法
const vector = new Vec2().addScalar(2).normalize();

onChange 回调函数

Vec2 类使用了 getter 和 setter，这使得每当向量的任一分量发生变化时，可以执行一个指定的回调函数：

// 创建一个新的 Vec2 向量并监听其变化
const vector = new Vec2().onChange(() => {
// 向量的一个分量刚刚发生了变化，执行某些操作...
const normalizedVector = vector.normalize();
});
// 这将会触发我们的 onChange 回调函数
vector.x = 1;
// 注意，这里再次修改分量会再次触发 onChange 回调！
vector.y = 2;

参数

  • X float，可选 X 轴的值。默认为 0。
  • Y float，可选 Y 轴的值。默认为 0。

属性

  • x (float): v7.0

    沿 X 轴的值。

  • y (float): v7.0

    沿 Y 轴的值。

方法

  • add(vector): v7.0

    将一个向量与当前向量相加。

    • vector Vec2 class object 要相加的向量。

    返回值：执行操作后的当前向量。

  • addScalar(scalar): v7.0

    将一个标量值添加到当前向量的每个分量上。

    • scalar float 要添加的数值。

    返回值：执行操作后的当前向量。

  • clone(): v7.0

    克隆当前向量。

    返回值：一个新的复制了当前向量的对象。

  • copy(vector): v7.0

    将另一个向量的值复制到当前向量中。

    • vector Vec2 class object 要复制的向量。

    返回值：复制完成后的新向量（即当前向量）。

  • dot(vector): v7.0

    计算两个向量之间的点积。

    • vector Vec2 class object 用于进行点积计算的向量

    返回值：表示两个向量点积结果的浮点数。

  • equals(vector): v7.0

    检查两个向量是否相等。

    • vector Vec2 class object 要比较的向量

    返回值：如果两个向量相等，则返回 true，否则返回 false。

  • max(vector): v7.0

    将给定向量中的最大分量值应用到当前向量中。

    • vector Vec2 class object 包含最大分量值的向量。

    返回值：应用最大分量后的向量。

  • min(vector): v7.0

    将给定向量中的最小分量值应用到当前向量中。

    • vector Vec2 class object 包含最小分量值的向量。

    返回值：应用最小分量后的向量。

  • multiply(vector): v7.1

    将当前向量与给定向量相乘（元素对元素）。

    • vector Vec2 class object 用于相乘的向量。

    返回值：执行乘法操作后的当前向量。

  • multiplyScalar(scalar): v7.1

    将当前向量的每个分量分别乘一个值

    • scalar float 用于相乘的数值。

    返回值：执行乘法后的新向量。

  • normalize(): v7.0

    将当前向量归一化为单位向量。

    返回值：已归一化的向量

  • sanitizeNaNValuesWith(vector): v7.0

    当向量中有 NaN 值时，使用指定向量中的相应值替换。主要用于内部处理以避免错误。

    • vector Vec2 class object 用作清理 NaN 值的参照向量。

    返回值：清理了 NaN 值之后的向量。

  • set(x, y): v7.0

    通过给定的 X 和 Y 参数设置向量的值。

    • x float 设置分量 X 的值
    • y float 设置分量 Y 的值

    返回值：设置了新值之后的向量。

  • sub(vector): v7.0

    从当前向量中减去另一个向量

    • vector Vec2 class object 用于相减的向量。

    返回值：执行减法操作后的当前向量。

  • subScalar(scalar): v7.0

    在每个分量上分别减去一个值

    • scalar float 减去的值

    返回值：执行减法操作后的新向量。

事件

  • onChange(callback): v8.0

    每当向量的 X 或 Y 分量发生变化时，执行一个函数。

    • callback function 回调函数

    返回值：返回当前 Vec2 对象本身，以便支持链式调用。
