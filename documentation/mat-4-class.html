Mat4 类

源码

基本使用

Mat4 是用于所有四维矩阵操作的类。

创建

要创建一个新的 Mat4（四维矩阵），你只需使用其构造函数。
如果不传递任何参数，它将创建一个四维单位矩阵。

// 创建一个新的 Mat4 单位矩阵
const matrix = new Mat4();

链式调用

由于 Mat4 类中大部分方法都会返回矩阵自身，因此你可以采用链式调用来连续操作：

// 创建一个新的 Mat4 矩阵并利用链式调用进行方法串联
const matrix = new Mat4().multiply(new Mat4()).scale(new Vec3(2, 2, 2));

参数

  • elements 长度为16的数组，可选 用于初始化矩阵的值。若不指定，则默认为一个4维单位矩阵。

属性

  • elements (Float32Array): v7.0

    一个长度为16的 Float32Array 数组，用于表示该矩阵。

方法

  • clone(): v7.2

    克隆出一个四维矩阵

    返回：新的四维矩阵

  • composeFromOrigin(translation, quaternion, scale, origin): v7.0

    通过四元数旋转、向量平移和向量缩放，围绕给定的原点创建一个矩阵。
    这种方法相当于分别应用平移、旋转和缩放矩阵，但执行速度更快。

    • translation Vec3 class object 平移向量
    • quaternion Quat class object 旋转四元数
    • scale Vec3 class object 缩放向量
    • origin Vec3 class object 围绕其进行缩放和旋转的原点向量

    返回：经过变换后的此矩阵。

  • copy(matrix): v7.0

    将一个矩阵复制到当前矩阵。

    • matrix Mat4 class object 要复制的 Mat4 类对象。

    返回：复制操作后的当前矩阵。

  • getInverse(): v7.1

    创建并返回一个当前矩阵的逆矩阵

    返回：当前矩阵的逆矩阵的新实例。

  • multiply(matrix): v7.0

    将当前矩阵与传入的参数矩阵进行乘法运算（矩阵组合）。

    • matrix Mat4 class object 用于乘法运算的 Mat4 类对象。

    返回：完成乘法操作后的当前矩阵。

  • scale(vector): v7.0

    根据提供的向量对当前矩阵应用缩放变换。

    • vector Vec3 class object 用于缩放的 Vec3 类对象。

    返回：完成缩放操作后的当前矩阵。

  • setFromArray(array): v7.0

    从给定数组设置当前矩阵的所有元素。

    • array array 包含至少16个元素的数组，这些元素会被用来填充矩阵。

    返回：设置数组元素后得到的当前矩阵。
