TextureLoader 类

源码

基本使用

TextureLoader 特别适用于在创建平面（Planes）或着色器通道（ShaderPasses）对象之前加载纹理资源。

您可以使用 TextureLoader 预加载网站中所有需要用到的纹理至GPU，同时展示加载进度指示器。这样一来，在预加载完成后启动应用程序时，就无需再为这些纹理资源创建新的加载实例，从而实现更快的应用初始化和流畅的用户体验。

创建

要创建一个新的纹理加载器，只需使用其构造函数。之后您将能够通过它的各种加载方法来加载媒体并创建纹理：

// “canvas” 是我们的 HTML 容器元素的 ID
const curtains = new Curtains({
container: "canvas"
});
// 创建一个新的纹理加载器
const loader = new TextureLoader(curtains);
// 加载一张图片
const image = new Image();
image.crossOrigin = "anonymous";
image.src = "path/to/my-image.jpg";
loader.loadImage(image, {
// 纹理选项（我们仅在此处设置其采样器名称）
sampler: "uTexture"
}, (texture) => {
// 纹理创建成功，做点什么
}, (image, error) => {
// 纹理创建失败的处理
});

参数

  • curtains Curtains class object Curtains 实例对象
  • crossOrigin string，可选 如果加载图片发生跨域时使用的跨域策略，默认为 "anonymous"。

属性

  • crossOrigin (string): v7.0

    用于加载媒体资源的跨域处理策略。

  • elements (array of objects): v7.0只读

    包含所有加载中和已加载元素的数组。每个元素都是一个包含纹理、其源以及用于加载该源的事件监听器的对象。

  • gl (WebGL context) v7.0只读

    WebGL 上下文

  • renderer (Renderer class object): v7.0只读

    由 curtains 对象创建的渲染器实例。

  • type (string): v7.0只读

    类的对象类型： "TextureLoader"。

方法

  • loadCanvas(canvasElement, textureOptions, successCallback): v7.0

    此函数接受一个HTML canvas元素，并使用它来创建一个Texture对象。

    • canvasElement HTML canvas element 要加载的HTML canvas元素。
    • textureOptions object，可选 用于该纹理的默认选项。请参阅 Texture 类的 参数。
    • successCallback function(texture)，可选 当源加载成功时执行的回调函数
      • texture Texture class object 新创建的纹理对象。在这里，您可以访问到纹理对象的 onSourceLoaded 和 onSourceUploaded 回调方法。

    示例： Text planes using canvas

  • loadCanvases(canvasElements, texturesOptions, successCallback): v7.1

    此函数接收一个HTML canvas元素数组，并为其中的每一个canvas元素创建一个Texture对象。

    • canvasElements array or collection of HTML canvas elements 要加载的一组HTML canvas元素。
    • texturesOptions object，可选 用于所有这些纹理的默认选项。请参阅 Texture 类的 参数。
    • successCallback function(texture)，可选 对于每个源成功加载时执行的回调函数
      • texture Texture class object 新创建的纹理对象。在这里，您可以访问到纹理对象的 onSourceLoaded 和 onSourceUploaded 回调方法。

  • loadImage(imageElement, textureOptions, successCallback, errorCallback): v7.0

    接收一个HTML图像元素（或图像源URL），并使用它创建一个Texture 对象。

    • imageElement HTML image element or string 图片元素或图片URL
    • textureOptions object，可选 应用到此纹理的默认选项。请参阅 Texture 类的 参数部分。
    • successCallback function(texture)，可选 当源加载成功时执行的回调函数
      • texture Texture class object 新创建的纹理对象。在这里，您可以访问到纹理对象的 onSourceLoaded 和 onSourceUploaded 回调方法。
    • errorCallback function(image, error)，可选 在源加载错误时执行的回调函数。
      • image HTML image element 尝试加载的图片
      • error object 抛出的错误信息

  • loadImages(imageElements, texturesOptions, successCallback, errorCallback): v7.1

    接收一个HTML图像元素数组（或图像源URL数组），并为其中的每个元素创建一个Texture对象。

    • imageElements array or collection of HTML image elements or strings 图片元素或图片URL的数组
    • texturesOptions object，可选 用于所有这些纹理的默认选项。请参阅 Texture 类的 参数。
    • successCallback function(texture)，可选 对于每个源成功加载时执行的回调函数
      • texture Texture class object 新创建的纹理对象。在这里，您可以访问到纹理对象的 onSourceLoaded 和 onSourceUploaded 回调方法。
    • errorCallback function(image, error)，可选 在每个源加载错误时执行的回调函数。
      • image HTML image element 尝试加载的图片
      • error object 抛出的错误信息

    示例： Asynchronous textures loading

  • loadSource(sourceElement, textureOptions, successCallback, errorCallback): v7.0

    接受一个源元素（或源URL），并使用它来创建一个Texture对象。

    • sourceElement either image, canvas or video HTML element or string 要加载的图像、canvas、视频HTML元素或源URL。
    • textureOptions object，可选 应用到此纹理的默认选项。请参阅 Texture 类的 参数部分。
    • successCallback function(texture)，可选 当源加载成功时执行的回调函数
      • texture Texture class object 新创建的纹理对象。在这里，您可以访问到纹理对象的 onSourceLoaded 和 onSourceUploaded 回调方法。
    • errorCallback function(source, error)，可选 在源加载错误时执行的回调函数。
      • source either an image, canvas or video HTML element 尝试加载但失败的源元素。
      • error object 抛出的错误信息

  • loadSources(sourceElements, texturesOptions, successCallback, errorCallback): v7.1

    此函数接受一个源元素数组（或源URL数组），并使用这些源创建多个Textures对象。

    • sourceElements array or collection of either images, canvases or videos HTML elements or strings 要加载的一组图像、canvas 或视频HTML元素或源URL。
    • textureOptions object，可选 应用到此纹理的默认选项。请参阅 Texture 类的 参数部分。
    • successCallback function(texture)，可选 对于每个源成功加载时执行的回调函数
      • texture Texture class object 新创建的纹理对象。在这里，您可以访问到纹理对象的 onSourceLoaded 和 onSourceUploaded 回调方法。
    • errorCallback function(source, error)，可选 在每个源加载错误时执行的回调函数。
      • source either an image, canvas or video HTML element 尝试加载但失败的源元素。
      • error object 抛出的错误信息

    此函数接收一个视频HTML元素（或视频源URL），利用它创建一个Texture对象

      • video HTML video element 尝试加载的视频

    • texturesOptions object，可选 用于所有这些纹理的默认选项。请参阅 Texture 类的 参数。
