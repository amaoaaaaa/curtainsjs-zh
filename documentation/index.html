curtains.js

用于为图像和视频制作动画的简单 WebGL 工具

这是什么？

现在，着色器已经成为实现网页上强大三维交互和动画效果的热门技术手段，在众多JavaScript库中，许多库能够处理WebGL，但在将3D网格与网页中的DOM元素进行相对定位时，往往给开发者带来一定的困扰。

Curtains.js 就是为了解决这个问题而创建的。它是一个小型普通 WebGL javascript 库，可将包含图像、视频和画布的 HTML 元素转换为 3D WebGL 纹理平面，允许您通过着色器对它们进行动画处理。 curtains.js 就是为了解决这一具体问题应运而生的。它能够将HTML元素中包含的图片、视频和canvas转换为带有纹理的3D WebGL平面，并允许开发者通过着色器对这些平面进行动画制作。

特别值得一提的是，你能够通过CSS来定义每个平面的大小和位置，这就使得在网页各处添加响应式WebGL平面变得极其简便易行。

此外，你还可以向场景中添加后期处理特效，以提升整个视觉表现力。

强烈建议查阅 文档 和示例，以便更好地了解其功能及所能实现的效果。

知识与技术要求

使用某个易用的WebGL库时，尽管其上手相对简单，但您当然需要具备扎实的HTML、CSS和JavaScript基础知识。

如果您之前从未接触过着色器（shader），那么您可能需要通过例如《 The Book of Shaders》这样的资源学习更多相关知识。在使用过程中，您必须理解什么是顶点着色器和片段着色器，了解如何运用uniform变量以及掌握GLSL基本语法。

安装

使用 npm：

npm i curtainsjs

加载ES模块：

import {Curtains, Plane} from 'curtainsjs';

请参阅文档以获取所有可用类的完整列表。

在浏览器中，您可以使用位于 dist 目录中的 UMD 文件：

<script src="dist/curtains.umd.min.js"></script>
// “canvas” 是我们的 HTML 容器元素的 ID
const curtains = new Curtains({
container: "canvas"
});
const plane = new Plane(curtains, document.getElementById("my-plane"));

在 React & Vue 中使用

如果您使用 React 或 Vue，您可能需要尝试 curtains.js 官方的 React 和 Vue 包： react-curtains 或 vue-curtains
