Quat 类

源码

基本使用

Quat 类用于旋转四元数的操作。四元数在三维空间中被用来表示旋转。

创建

要创建一个新的 Quat（四元数），只需使用其构造函数。
如果不传递任何参数，它将创建一个代表绕 X、Y 和 Z 轴旋转角度为 0 的四元数，并且轴顺序为 "XYZ"：[0, 0, 0, 1]。

// 创建一个新的四元数
const quaternion = new Quat();

链式调用

由于 Quat 类中的大多数方法都会返回四元数自身，因此你可以利用链式调用来连续执行多个操作：

// 创建一个新的 Quat 四元数并利用链式调用进行方法串联
const quaternion = new Quat().setAxisOrder("YXZ").setFromVec3(new Vec3(1, 1, 1));

参数

  • elements 长度为4的浮点数组，可选 用于初始化四元数的值。默认为 [0, 0, 0, 1]，表示无旋转。
  • axisOrder string，可选 用于旋转计算的轴顺序，默认为 "XYZ"。

属性

  • elements (Float32Array): v7.0

    一个长度为4的浮点数数组，以 Float32Array 类型表示四元数。

  • axisOrder (string): v7.0

    用于三维旋转计算的轴顺序。

方法

  • clone(): v7.2

    克隆该四元数

    返回：新克隆的四元数。

  • copy(quaternion): v7.0

    将一个四元数复制到当前四元数中

    • quaternion Quat class object 要复制的四元数

    返回：复制操作后的当前四元数。

  • equals(quaternion): v7.0

    检查两个四元数是否相等（即它们具有相同的数值和轴顺序）

    • quaternion Quat class object 用于比较的四元数

    返回：如果两个四元数相等，则返回 true，否则返回 false。

  • setAxisOrder(axisOrder): v7.0

    设置四元数的轴顺序

    • axisOrder "XYZ" | "XZY" | "YXZ" | "YZX" | "ZXY" | "ZYX" 表示要使用的轴顺序的字符串

    返回：设置轴顺序后的当前四元数。

  • setFromArray(array): v7.0

    从数组设置四元数的数值

    • array array 至少包含 4 个浮点数的数组

    返回：设置数值后的当前四元数。

  • setFromVec3(vector): v7.0

    使用欧拉角（以 Vec3 对象形式）和其轴顺序设置旋转四元数

    • vector Vec3 class object 作为旋转源的 Vec3 向量

    返回：应用旋转后得到的当前四元数。
